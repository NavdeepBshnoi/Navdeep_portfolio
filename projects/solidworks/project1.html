Sketch Parts ReadMe

Step 1 – Sketching

Initial Design Phase

In this phase, 2D sketches for each prosthetic component were created in SolidWorks. Geometrical relations, constraints, and dimensions were defined for precision and easy modification.

Step 2: Parts Modeling

Component Creation

Each part of the prosthetic arm, including joints and connectors, was modeled with assigned material properties and appearance textures.

Step 3: Assembly

Assembly Overview

The modeled parts were assembled using various mates (coincident, concentric, parallel). Motion simulation was performed to verify range of movement and avoid interference.

Step 4: ReadMe / Conclusion

This project demonstrates the complete SolidWorks workflow from 2D sketching to 3D assembly. Future work includes actuator integration and motion analysis coupling with ANSYS.
